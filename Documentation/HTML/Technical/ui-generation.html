Black, White & Gold Color System

The MTM WIP Application uses a sophisticated black, white, and gold color scheme for maximum readability and professional appearance:

Primary Colors

<!-- Primary Colors --> Black: #000000 - Primary text, borders, icons White: #FFFFFF - Background, secondary text Gold: #FFD700 - Primary accent, buttons, highlights Dark Gold: #FFA500 - Hover states, secondary accents Light Gold: #FFFF99 - Disabled states, subtle backgrounds Silver: #C0C0C0 - Tertiary accents

Usage Guidelines

  • Gold (#FFD700): Primary buttons, active states, navigation highlights
  • Dark Gold (#FFA500): Hover effects, secondary actions
  • Black (#000000): Primary text, icons, borders
  • White (#FFFFFF): Backgrounds, contrast text

Resource Mapping

<!-- Avalonia Resource Definitions --> <SolidColorBrush x:Key="PrimaryBrush" Color="#FFD700"/> <SolidColorBrush x:Key="AccentBrush" Color="#FFD700"/> <SolidColorBrush x:Key="SecondaryBrush" Color="#FFA500"/> <SolidColorBrush x:Key="BackgroundBrush" Color="#FFFFFF"/> <SolidColorBrush x:Key="TextBrush" Color="#000000"/> <!-- Gradient brushes for impact --> <LinearGradientBrush x:Key="HeroGradientBrush" StartPoint="0,0" EndPoint="1,1"> <GradientStop Color="#000000" Offset="0"/> <GradientStop Color="#FFD700" Offset="0.5"/> <GradientStop Color="#FFA500" Offset="1"/> </LinearGradientBrush>

View Structure Standards

üìê Mandatory Proportional Scaling

ALL Views MUST implement proportional scaling to maintain consistent appearance across different window sizes.

Implementation Pattern

<!-- MANDATORY: Viewbox wrapper for proportional scaling --> <Viewbox Stretch="Uniform" StretchDirection="Both"> <!-- Fixed dimensions for scaling reference --> <Grid Width="800" Height="600"> <!-- Content scales proportionally --> </Grid> </Viewbox>

AXAML Structure Standards

<UserControl xmlns="https://github.com/avaloniaui" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:vm="using:YourApp.ViewModels" x:Class="YourApp.Views.YourView" x:DataType="vm:YourViewModel" x:CompileBindings="True"> <Viewbox Stretch="Uniform" StretchDirection="Both"> <Grid Width="800" Height="600"> <!-- Your content here --> </Grid> </Viewbox> </UserControl>

Layout Principles

  • Use Grid for complex layouts with rows/columns
  • Use StackPanel sparingly, prefer Grid for performance
  • Default margins: Margin="8" for containers
  • Card padding: Padding="24" for spacious content
  • Gold borders: BorderBrush="#FFD700" BorderThickness="2"
  • Box shadows: BoxShadow="0 4 8 0 rgba(255,215,0,0.3)"
