üìã Key Patterns & Practices

üîß Dependency Injection

Use AddMTMServices() extension method for service registration. Never register services individually.

üóÑÔ∏è Database Access

All database operations must use stored procedures via Helper_Database_StoredProcedure.

    connectionString, "sp_ProcName", parameters);

‚ö° ReactiveUI Patterns

Use RaiseAndSetIfChanged for properties and ReactiveCommand for commands.

set => this.RaiseAndSetIfChanged(ref _field, value);

üéØ Transaction Types

TransactionType is determined by USER INTENT, not operation numbers.

TransactionType.IN   // Adding stock
TransactionType.OUT  // Removing stock
TransactionType.TRANSFER // Moving stock
