MTM-Specific Model Features

üîÑ Transaction Types

Critical Rule: TransactionType is determined by USER INTENT, not Operation numbers

IN: User adding inventory (regardless of operation)
OUT: User removing inventory (regardless of operation)
TRANSFER: User moving between locations (regardless of operation)

üè∑Ô∏è Operation Numbers

Operation numbers ("90", "100", "110") are workflow step identifiers, NOT transaction type indicators

They represent manufacturing/processing stages and help track production progress without determining inventory flow direction.
